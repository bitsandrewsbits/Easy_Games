##### To start game:
###### 1) Work dir - game dir
###### 2) Create venv: python3 -m venv "your_venv_name"
###### 3) Activate venv: source your_venv_name/bin/active
###### 4) Install libraries: pip install -r requirements.txt
###### 5) Start the game: python game.py
